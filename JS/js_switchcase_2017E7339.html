switch-case(조건문)

 switch-case 조건문은 switch(){ case : ...; break;}가 기본 형태로 조건식에 값에 여부에 따라 해당되는 실행문을 실행하도록 하는 문장이다.
값에 따라 서로 다른 코드를 실행할 때, switch 문이 적합하다.	
	
		switch(식) {
            case 값1: // 식의 결과가 값1과 같을 때
                실행 문장 1;
                break;
            case 값2: // 식의 결과가 값2와 같을 때
                실행 문장 2;
                break;
                ...
            case 값m:
                실행 문장 m; // 식의 결과가 값과 같을 때
                break;
            default: // 어느 값과도 같지 않을 때
                실행 문장 n;        
	
case문 
1. case 문의 '값'은 상수 (리터럴)만 가능 하다.  
        case 1 : 
        case 2.7 :
        case “Seoul” :
        case true :

2.case 문의 '값'에 변수나 식은 사용 불가 
        case a :		// 오류. 변수 a 사용 불가
        case a > 3 :	// 오류. 식(a>3) 사용 불가
    
break문 
switch 문 종료 : break; 문을 만날 때까지 아래로 코드 계속 실행
