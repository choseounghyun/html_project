TABLE(표만들기) 
표를 작성할 수 있게 도와주는 태그.
      전체 구성은 <table>, <caption>, <thead>, <tfoot>, <tbody>, <tr>, <td>, <th> 등으로 구성된다. 
        

    <table> : 표 전체
    <caption> : 표 제목
    <thead> : 표 헤드 셀 그룹
    <tfoot> : 표 바닥 셀 그룹
    <tbody> : 표 데이터 셀 그룹
    <tr> : 행. 여러개의<td>, <th> 포함
    <td> : 데이터 셀
    <th> : 제목(헤딩) 셀
