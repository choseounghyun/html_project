SELECT(콤보 박스 만들기) 
콤보박스는 드롭다운 목록을 보여주고 그중 하나를 선택하는 폼요소이다. 
select태그는 콤보박스 전체를 표현하고 option태그는 항목 하나를 표현한다.

        사용 법 : <select name="요소이름" size="개수(창 크기, 보일수 있는 최대 항목)" multiple(이 속성이 있으면 다수 항목 선택 가능)></select>
        사용 법 : <option value="옵션 값" selected(이 속성이 있으면 초기에 선택 상태로 출력)></option>   
        코드 예시 
            <select name="china">
                <option value="1">짜장면</option>
                <option value="2" selected>짬뽕</option>
                <option value="3">탕수육</option>
            </select>
