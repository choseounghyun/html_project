BUTTON/SUBMIT/RESET(폼 요소 : 버튼) 
BUTTON
단순 버튼을 만드는 form요소 중 하나다. 
        사용 법 : <input type="button" value="버튼에 출력되는 문자열">
        <button type="button">버튼</button>
        
        예시 
        
        
        
SUBMIT
폼 데이터를 웹 서버로 전송한다.
        사용 법 : <input type="sumit" value="버튼에 출력되는 문자열">
            <button type="submit">전송</button>
        
        예시
        
        
        
RESET
폼에 입력된 내용을 모두 지우고 초기화하는 버튼이다.
        사용 법 : <input type="reset" type="버튼에 출력되는 문자열">
            <button type="reset">리셋</button>
        
        예시
