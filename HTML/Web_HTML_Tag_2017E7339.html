기본태그

기본 태그
  • <!DOCTYPE html>
    • html5임을 브라우저에게 지시하는 태그
  • <head>
    • css스타일, 자바스크립트,메타태그들이 위치
  • <body>
    • 자바스크리트 코드를 포함 할 수 있으며 HTML 구조 및 내용이 들어간다.
  • <head>안에 들어가는 태그
    • <meta>
      • 본 문서의 키워드 폭은 제목, 요약을 나타내는 태그로서 활용
    • <title>
      • 문서의 제목을 나타내는 태그
    • <link>
      • 주로 CSS파일을 연결할 때 사용하는 태그
    • <style>
      • html 문서안에css그대로 사용하고자 할때 적어주는 태그
  • <body>안에 들어가는 태그
    • 제목 태그<h숫자>
      • 본문의 헤드라인 요소
    • <p>
      • 문단 태그로서 텍스트를 감싸게 되는 태그, 명확한 의미를 위해 사용
    • <pre>
      • 형식화된 텍스트를 나타내는 태그, 감싸져있는 띄어쓰기,들여쓰기,줄바꿈 등이 그래도 웹페이지에 표시
    • <br>
      • 줄바꿈 태그로서 사용
    • <hr>
      • 장면의 전환, 다른 화제로의 전환 등을 위해 사용
    • 강조태그<strong><em>
      • <strong>은 글자를 굵게 <em>은 글자를 기울게 표시
    • <a>
      • 하이러링크 태그
    • <img>
      • 이미지 출력태그
    • <div>
      • 레이아웃을 짜게되는 태그<header>,<section>,<article>,<footer>,<nav>,<aside>등이 있다.
    • <table>
      • 표를 만드는 태그<thead>,<tbody>,<tfoot>이 있으며 <tr>은 행을 <td>는 열을 나타낸다.
