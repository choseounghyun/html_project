CHECKBOX/RADIO(폼 요소 : 체크 박스와 라디오 버튼) 
CHECKBOX
체크박스는 선택/해제 중 하나를 선택하는 폼이다. 
        사용 법 : <input type="checkbox" value="요소 값(선택 상태일 때, 웹서버에 전송되는 값)" checked(초기에 선택 상태로 출력)>
체크 박스
짜장면 짬뽕 탕수육 
RADIO
라디오버튼은 name 속성 값이 같은 라디오버튼들이 라나의 그룹을 형성하고,
그 중 하나만 선택 되는 폼 요소이다.    
        사용 법 : <type="radio" size="개수(콤보박스 창 크기(보일수 있는 항목 개수)디폴트 1)" multiple(이 속성이 있으면 다수 항목 선택 가능)>
