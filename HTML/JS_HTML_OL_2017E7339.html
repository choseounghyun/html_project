OL(순서있는 리스트)

데이터를 나열할 때 순서를 나타내여 사용되는 태그이다.

<ol>태그를 사용 사용법 : <ol type="1|A|a|I|i" start="value"><li>리스트</li></ol> 

            마커종류(type) :"1"(디폴트), "a", "A", "I", "i" 
            시작값(start) 
            <ol>
                <li>리스트1</li>
                <li>리스트2</li>
                <li>리스트3</li>
            </ol>
           
            <ol>태그 적용</ol>
            
    
                
  1. 리스트1
    2. 
                
  2. 리스트2
    3. 
                
  3. 리스트3
    4.
