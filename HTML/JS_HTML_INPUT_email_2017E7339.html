eMAIL/URL/TEL(폼 요소 : 이메일, url, 전화번호 입력받기) 
email 주소, URL, 전화번호와 같이 텍스트가 특정 형식에 맞게 입력되었는 지를 검사하는 폼 요소이다. 
eMAIL 
메일 주소만 받는 폼요소 
        사용 법 : <input type="email" placeholder="id@host(형식)">
           
URL
'http://www.anver.com'과 같이 W3C의 규정에 맞도록 입력받는 폼요소
        사용 법 : <input type="url" placeholder="http://(형식)">
        
TEL
전화번호만 입력받는 폼요소 
        사용 법 : <input type="tel" placeholder="010-1234-5678(형식)">
