SELECTOR CLASS 
class값을 선택하는 셀렉터입니다. class는 html문서에서 중복되어 쓸 수 있습니다.

사용법 : .class이름

예시
    <head>
        <style>
          /* class 어트리뷰트 값이 container인 모든 요소를 선택 */
          /* color 어트리뷰트는 자식 요소에 상속된다. */
          .container { color: red; }
          /* not supported in IE */
          #p2 { color: initial; }
        </style>
      </head>
     
      <body>
        
        <div class="container">
          <p id="p1">paragraph 1</p>
          <p id="p2">paragraph 2</p>
        </div>
      </body>
